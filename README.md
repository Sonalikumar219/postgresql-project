# PostgreSQL Sales Management Project

This project demonstrates a simple sales database system using PostgreSQL.

## Project Description
The project contains tables for:
- Customers
- Products
- Orders
- Order Items

It shows how to design a database and write SQL queries for business analysis.

## Files in this project
- create_tables.sql : Creates all required tables
- insert_data.sql : Inserts sample data into tables
- queries.sql : Contains SQL queries for analysis

## Skills Used
- CREATE TABLE
- INSERT INTO
- JOINS
- GROUP BY
- HAVING
- Aggregate Functions (SUM, COUNT)

## How to Run
1. Create a database in PostgreSQL
2. Run create_tables.sql
3. Run insert_data.sql
4. Run queries.sql

## Output Examples
- Total sales per customer
- Best selling product
- Orders by city
